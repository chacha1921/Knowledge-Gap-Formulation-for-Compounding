Slide 3 of 7 — How a VLM Processes an Image: Architecture Source: Dosovitskiy et al., An Image is Worth 16×16 Words — ICLR 2021

Vision Transformer (ViT) — From Image to Token Sequence

Three Components Working in Sequence

Vision Encoder

Translates raw image patches into vectors — the same number-language the LLM already understands from text.

Projection Layer

Thin trainable bridge that rescales vision encoder output to exactly match the LLM's embedding dimension.

Language Model (LLM)

Receives prompt text + all patch vectors in one combined list. Attends across all tokens equally — no special treatment for image vs text.

Slide 4 of 7 — Why Wide Tables Fail — Now I Can Predict It Tile boundary = row continuity break
Each 512 × 512 tile covers ~3.4 × 3.4 inches of the page. An A4-wide table (8.27 inches) spans 2–3 tiles. Tiles are independent crops — no shared positional encoding across tile boundaries. A row that straddles a boundary appears as two disconnected groups of patches in the attention sequence.

Tables that succeed

PropertyWhy
Width < 3.4 inchesFull table fits in one tile — spatial layout intact
Dense, single-columnCells don't cross tile boundaries
Page centreNot split by edge tile boundaries

Tables that fail

PropertyWhy
Width > 3.4 inchesRow straddles tile — row continuity lost
Near page edgeContent split by margin tile — partial rows
Multi-column wideColumns in different tiles, no linking signal
Slide 5 of 7 — The Fix: SmartBBoxCropper 15-line OpenCV preprocessing · committed to Document-Intelligence-Refinery

The Insight

If the page is cropped to its content bounding box before the VLM call, margins are removed, the image is smaller (fewer tiles), and tables are more likely to fall within a single tile. Follows directly from understanding the tile structure.

import cv2, numpy as np

def smart_bbox_crop(img_bytes: bytes, padding: int = 20) -> bytes:
    nparr  = np.frombuffer(img_bytes, np.uint8)
    img    = cv2.imdecode(nparr, cv2.IMREAD_COLOR)
    gray   = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
    _, thr = cv2.threshold(gray, 250, 255, cv2.THRESH_BINARY_INV)  # find dark content
    coords = cv2.findNonZero(thr)
    if coords is None: return img_bytes                          # blank page — skip
    x, y, w, h = cv2.boundingRect(coords)
    ih, iw = img.shape[:2]
    x, y = max(0, x-padding), max(0, y-padding)
    w, h = min(iw-x, w+2*padding), min(ih-y, h+2*padding)
    _, buf = cv2.imencode('.png', img[y:y+h, x:x+w])
    return buf.tobytes()

img_bytes = smart_bbox_crop(img_bytes)   # added before every VLM call
